## Controls

Keyboard | Joystick | Command
:------- | :------- | :------
`W`/`S` or `UP`/`DOWN` | `UP`/`DOWN` | Move Paddle
`ENTER` | `START` | Start/Pause Game
`F11` | `Y` | Toggle Fullscreen
